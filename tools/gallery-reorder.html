Gallery Reorder Tool cambios sin guardar

Arrastra las imágenes para cambiar el orden. La primera imagen será la hero de la ficha.

Cómo usar:
1. Abre esta página localmente (doble clic en el archivo).
2. Selecciona el manifest.json del barco que quieres reordenar.
3. Arrastra las imágenes al orden deseado.
4. Haz clic en "Guardar orden" para descargar el manifest actualizado.
5. Reemplaza el manifest.json original con el descargado.
6. Ejecuta: python scripts/integrate_galleries.py para regenerar las galerías HTML.
7. Despliega con: bash deploy/deploy-staging.sh
